4. What are arguments and parameters? What is the difference between them?
    Parameters are variables declared in a function's definition and act as placeholder for the values the
    function expects to receive
    Arguments are the actual expressions or values that are passed to a function when you call it
    Difference is that parameters are defined in the functions declaration to specify what kind of data the function
    expects while arguments are the values that are passed when calling a function.
